#### 地图接入

1. 下载文件包 [GIS 剑谱](http://192.168.120.121:6012/gis/#/main/gisAccess)
2. 在 html 文件中引入`地图依赖 leaflet.map.min.js` 和`地图配置 mapConfig.js`
3. 利用 httpVueLoader 注册地图组件，绑定到页面
